Collision

In this lesson we will cover elastic collision, if you haven't read the previous lessons be sure to check those out as they cover some basic concepts that we will cover here. There are two parts of studying collisiions: collision detection and what to actually do what two objects collide. Collision detection doesn't involve any physics but rather geometery. Two circles are considered to be colliding when they overlap. An easy way to calculate this is to find the distance between the centers of the two circles (a function for this is usually included in graphics libraries but you can find an equation for it on wikipedia if you don't already know it) and check if the summation of their radii is greater than that distance. This is easy to visualize if you think about a Venn diagram. As for elastic collision between two objects, if they have the same mass you can simply switch their velocities so that the first object has the old velocity of the other object and vice versa.
