About the site

This project was created as a class project for San Francisco State University CSC412 where we were able to make a website on our interests. I chose this topic as I have loved video games my whole life. I'm hoping that by making this site, I can help others begin programming projects that interest them by removing one of the major non-programming hurdles of physics and math, as we often don't need the full complexity of them to code simple games. This site is currently incomplete and open to contributions on github. If you have any questions or suggestions please leave a pull request.
