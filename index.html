Video Game Physics

The basics of physics needed to start coding video games.

Why?

While you can create video games without using physics, your risk your game feeling cluncky. Physics is what gives games life. While they can be difficult it will pay off in the long run to have a better-feeling game. The goal of this site is to give a straightforward guide on using basic physics that you will most likely need to program your game. For the sake of being language-agnostic, there won't be any demonstration code on how this is done, but there should be enough details for you to be able to apply what you learn. To be able to follow the lessons, you should have a basic understanding of algebra and programming so that you can apply these lessons. In each lesson, there will be an explanation, some simple algebraic formulas, and an interactive animation to test how this concept is used. All forms of calculus have been ommited in order to make this guide more accessible, but it may be necessary for more complex or specific problems that must be resource-efficient. For beginners who have little programming experience and don't which programming language or platform to use, I would highly recommend p5.js, a javascript library as it is easy to use and what is used for the examples on this site.
